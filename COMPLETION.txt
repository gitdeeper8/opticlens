1️⃣  System Setup:
    $ python -m venv .venv
    $ source .venv/bin/activate
    $ pip install -r requirements.txt
    $ pre-commit install

2️⃣  Verify Installation:
    $ opticlens doctor
    $ python scripts/validate_environment.py

3️⃣  Test Mie Scattering:
    $ python scripts/test_mie.py --radius 0.1 --wavelength 0.55

4️⃣  Test Refractive Index:
    $ python scripts/test_edlen.py --pressure 101325 --temp 288.15

5️⃣  Fetch Real-time Data:
    $ opticlens data fetch --source aeronet --station GSFC --hours 24

6️⃣  Launch Dashboard:
    $ opticlens dashboard --port 8501
    $ # Open: https://opticlens.netlify.app

7️⃣  View Documentation:
    $ cd docs/ && make html
    $ # Open: docs/build/html/index.html
